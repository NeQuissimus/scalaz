Set

Scalaz provides instances for Scalaâ€™s built-in Set such as Eq.

Typical imports

import scalaz.Scalaz._
import scalaz.std._

Use the type class instances

scala> val set1: Set[Int] = Set(1, 2, 3)
set1: Set[Int] = Set(1, 2, 3)

scala> val set2: Set[Int] = Set(1, 2, 3, 3)
set2: Set[Int] = Set(1, 2, 3)

scala> set1 === set2
res0: scalaz.Boolean = true
